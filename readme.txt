Zingiri Web Shop is a Wordpress plugin that adds fantastic ecommerce capabilities to a great content management system.

== Description ==

Zingiri Web Shop is a Wordpress plugin that turns a great content management system into a fantastic e-commerce solution.
 
The Live version is a special edition of our standard plugin that uses web services stored on Zingiri's servers. 
It makes installation a breeze and ensures you are always up to date with the latest versions.
Read more in the plugin's FAQ about what that means.

The main features are:

* Simple installation & easy configuration
* Sell digital and physical products
* Full customer management module
* Complete ordering and stock management module
* Ajax powered shopping cart & one page checkout
* Support for payment portals like Paypal (Standard & Express), Authorize.net and others to come
* Multiple language libraries included: English, Dutch, French, German, Spanish, Portuguese, Danish, Estonian, Finish, Greek, Hungarian, Italian, Japanese, Norwegian, Polish, Romanian, Russian, Swedish, Slovak, Thai, Turkish and Serbian
* Localized tax support: including support for complex cascading tax systems 
* Easily migrate your current web shop by uploading your products via an XML file

== Installation ==

Simply install like any other Wordpress plugin.

== Frequently Asked Questions ==

= This plugin uses web services, what exactly does that mean? =
Web services are simple way of delivering software solutions. Basically it means that the software & data is hosted on our secure servers and that you can access it from anywhere in the world. 
No need to worry about backing up your data, managing systems, we do it for you.

= What about data privacy? =
Web Shop Live uses web services stored on Zingiri's servers. In doing so, personal data is collected and stored on our servers. 
This data includes amongst others your admin email address as this is used, together with the API key as a unique identifier for your account on Zingiri's servers. 
We have a very strict [privacy policy](http://www.zingiri.com/privacy-policy/ "privacy policy") as well as [terms & conditions](http://www.zingiri.com/terms/ "terms & conditions") governing data stored on our servers.

== Upgrade notice ==

Upgrades are handled automatically. Simply upload the latest version, go to settings and select Update. 

Before upgrading, make sure you back up your database first!
